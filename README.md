# AI Dashboard - Restaurant Tips Analysis

## Project Overview

This project is an interactive AI Dashboard built using Streamlit, Pandas, NumPy, and Plotly.

The dashboard analyzes restaurant customer tipping behavior and provides interactive insights through charts, KPIs, filters, and business observations.

## Features

* Dataset Overview
* Data Cleaning
* Key Performance Indicators (KPIs)
* Interactive Filters
* 5 Data Visualizations
* Business Insights

## Dataset Information

Dataset: Restaurant Tips Dataset

Columns:

* total_bill
* tip
* sex
* smoker
* day
* time
* size

## Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Streamlit

## Installation

Install required packages:

pip install -r requirements.txt

## Run Project

streamlit run app.py

## Business Insights

* Higher bills generally produce higher tips.
* Weekend days generate more restaurant revenue.
* Customer tipping behavior varies by gender and smoking status.
* Interactive filters help analyze customer segments.
